Creating a video kiosk on Windows 10 Home
Welcome!

Why set up a Kiosk this way?
...

Windows 10 introduced a Kiosk feature, which can lock the user into a specific app. However, this does have a couple limitations:

  1. It only works with UWP Apps (so your video player choices are very limited)
  2. You need to have Windows 10 Pro, Enterprise, Education or Windows 11

This is usually not well suited for students or hobbyists wanting to set up a quick, looping, video display without having to pay for the Pro edition of Windows.

Setting it all up
...

1. MPV
...

MPV is a free and open source video player. The main reason I'll be using it is because of it's powerful configuration file, allowing me to turn it into the perfect video kiosk.

To download MPV together with my configuration, download zyphen-mpv-kiosk.zip.

I'm using the portable mpv build provided by shinchiro from mpv.io.

Tip

Alternatively, you can get MPV yourself and place the following contents in your mpv.conf file:

loop-playlist
fullscreen
no-input-cursor
no-input-default-bindings
no-osc

So far, you've got a video player that's:

  • Fullscreen by default
  • Has no mouse or keyboard inputs
  • Has no OSD (on screen display)

That seems pretty secure, right? Well, not quite. Anyone can still press the Windows key and open any other app or even close down MPV from the taskbar or with the Alt+F4 shortcut.
To fix this, we'll need the help of another tool, this time one developed by Microsoft.

2. Microsoft PowerToys
...

Microsoft PowerToys is a set of utilities for power users to tune and streamline their Windows experience for greater productivity.

learn.microsoft.com

2.1. Getting PowerToys
...

You can get the latest PowerToys installer from the official github page.

Per user installation

Using the "Per user" version shouldn't require Administrator Privileges to install, but might not always work properly.

2.2. Configuring Always On Top
...

From the Always On Top tab in the PowerToys app, turn on Enable Always On Top. I also recommend unchecking the box next to Do not activate when Game Mode is on.

Warning

If you plan on disabling the Windows key by following the next step, make sure to change your 'Activation Shortcut' to one that doesn't include it.

2.3. Disabling the Windows key
...

In order to disable the use of the Windows button we'll be using a module called Keyboard Manager, a tool for remapping keys.

From the Keyboard Manager tab in PowerToys, enable the module by turning on Enable Keyboard Manager. Now, you can click on Remap a key.

From this window, select both Win (Left) and Win (Right) as Physical Keys and Disabled as Mapped to.

Danger

This will fully disable both of your Windows keys until you remove the mappings from this same menu.
If you chose to install PowerToys Machine wide, this will be applied even after a reboot and across all users.

2.4. Disabling Alt+F4
...

In order to disable the Alt+F4 key combination that closes apps on Windows, we'll be using the Remap a shortcut function of the Keyboard Manager module.

This is very similar to the key remapper, and we'll just need to set the Alt+F4 Physical Shortcut to Disabled. Once again, you'll need to do this for both Alt (Left) and Alt (Right).

Danger

After this, the only way to quit MPV is to restart your computer through the Ctrl+Alt+Del screen. This could be disabled using the same method as Alt+F4, but I don't recommend doing so.

3. You're done!
...

Finally, drag and drop your video onto mpv.exe and enjoy your video kiosk.

Tip

You can even drag and drop multiple files over mpv.exe, and they will be played as a playlist!

Limitations
...

My aim is to create a video kiosk for Windows 10 Home, which means not using the Kiosk (Assigned access) feature.
This means we can't truly lock down Windows, and there will probably always be some type of way to get around these messures (if you can think of any, let me know at contact@zyphenvisuals.com). A couple already known limitations are:

  • If Task Manager is set to always display on top, it will still appear above your video player. If this is a concern, you can block the Ctrl+Shift+Esc shortcut, though Ctrl+Alt+Del cannot be disabled.
  • If someone focuses an app in the background and presses the Always On Top activation shortcut set earlier, that window will appear over MPV. If this is a concerc, you can block the Alt+Tab and Win+Tab shortcuts, as well as any other shortcuts that open apps (which would grab focus by default). This is however almost never a concern, since the person would need to know exactly what to do and what keys to press to get another window to appear on the screen.

Additionally, this setup will not autostart after a reboot. This could be configured by a .bat script when the user logs in, but for my purposes that went a bit too far.
