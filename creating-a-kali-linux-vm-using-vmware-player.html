Welcome!

This article was prompted by a recent school assignment, where we needed to use the package pwtools. This is a pretty common occurrence in the world of CTF competitions, but is not officially supported on Windows, forcing everyone's hand into using Linux.

But what is Linux?
...

Linux is an operating system, just like Windows or MacOS. While it's use in the desktop world is still limited, programmers should get familiar with it since it powers a staggering 96.3% of the top one million web servers.
One of the things that makes Linux so different from both Windows and MacOS is that there isn't just one variant of it. In fact, there are over 600 different distributions (distros) of Linux, each with it's own look, features and use cases.

So why Kali?
...

Kali Linux is a Linux distro designed with penetration testing in mind. This means it will come pre-installed and configured with a whole set of networking and engineering tools that can prove useful in solving CTF challenges.
For this guide we'll be installing Kali using the official installer image available here. I will be using the NetInstaller version, which will download most of the OS while installing it later.

How are we gonna install it?
...

Since we don't want to have to abandon our original operating system, we can leverage a tool known as virtualization. In simple terms, we'll install a program that can run an operating system like any normal application, creating a sort of fake computer inside our real machine.
Today we'll be using the free VMWare Player, which can be downloaded from here. In order to be able to run this app, the important requirements are the following:

SpecificationMinimum
Host Operating SystemWindows 8
Ram4GB
Storage20GB Free

Creating the Virtual Machine
...

Once you have VMWare Player installed, go ahead and click Create a new Virtual Machine.

On the first screen select Installer disc image file (iso) and find where you saved your installer.

Next, select your Guest operating system as Linux and Other Linux 5.x kernel 64-bit.

The chosen Location will determine where all of the files inside the virtual machine will be stored.

Careful!

This file can become huge, as it contains both the operating system and every file downloaded inside of the virtual machine

Next, you can choose how much space to allocate your new virtual machine. I recommend using at least 20GB, but if you know you'll install many apps inside of it you can definitely go higher. I also recommend ticking Store virtual disk as a single file.

Allocating the resources
...

Now we have to allocate some hardware resources to our Virtual Machine, which it will (in a sense)
take away from our day to day operating system while it's running

For Memory, I recommend using at least 2GB of RAM. If you plan on doing heavy multitasking or compiling inside of the VM, you can go higher with this number.

Warning

If your host Operating System is Windows, make sure to leave at least ~4GB of RAM for it to use.

For Processors, use at least 2 cores. You can increase this amount to further speed up the VM, but I recommend always leaving at least 2 cores for your host Operating System.

Warning

This number actually refers to threads, not necesairly physical cores. If your CPU uses hyperthreading, your number of available threads might be double your amount of cores.

Finally, go to the Display tab and enable Accelerate 3D graphics, which will make the desktop of the virtual machine smoother. I recommend setting the Graphics memory to no more than 512MB, since this memory is taken from the Memory we allocated earlier. Obviously, if you allocated more memory, you can increase this further if you need higher resolutions.

Installing Kali Linux
...

You can now boot up your Kali Linux Virtual Machine, select Graphic install and go through the guided install process, where most settings can be left to default.

Desktop Environment
...

Unlike Windows or MacOS, Linux lets you choose the desktop enviornment yourself, letting you choose your own look and workflow.

XFCE - Speed
...

XFCE is the default desktop environment for Kali Linux. It is extremely lightweight and customizable, making it perfect for lower power machines.

Gnome - Simplicity
...

Gnome is a simple yet stylish desktop environment with it's own opinionated workflow, making it perfect for people wanting to try something new and fresh.

KDE - Customization
...

KDE is an incredibly feature-packed and customizable desktop environment with a familiar Windows-like default layout, perfect for power uses wishing to customize every aspect of their machine.
